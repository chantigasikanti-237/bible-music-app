04 · Interaction flow
How the four core actions resolve

Search

Keystroke → 400ms debounce → GET /api/audio/search scoped to active language → Related (first 6) + full Results list.

Tab switch

Tap pill → local state flips → shared layoutId springs the active pill → grid content cross-fades via AnimatePresence.

Favorite

Heart tap → toggled in-memory → synced to localStorage["music_favorites_v2"] → Downloads tab reads the same key, no API round-trip.

Play

Play tap → GET /api/audio/url/:videoId resolves a stream URL → loaded into an HTMLAudioElement → mini player mounts above the bottom nav.
